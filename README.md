# Puoppo 🕊

## 🚀 概要：なぜ「Puoppo」なのか

**「現場には、現場の最適解がある。」**

IT開発系の資格なし、プログラミング実務未経験のド初心者が、基礎学習からわずか42時間(開発期間は1日(10時間))で制作にありついた、AIトレンド自動分析システム――それが『Puoppo（ポッポ）』です。

特定の流行を追う一発芸のようなシステムではなく、「どんな現場のデータも制御できる」という、長年履き込んでも壊れないデニムのような普遍的な設計思想で骨格を仕立て上げました。

## 💡 特徴と圧倒的なタイパ

- **爆速の自動収集＆要約**
  テキストボックスに「知りたい情報（例：政治・物価高、最新音楽ニュース等）」を打ち込むだけ。AIがインターネット上の最新記事を100件レベルで超高速収集し、わずか10秒で濃厚に要約してPC・スマホに滑り込ませます。無駄なコメンテーターの意見をダラダラ聴く必要はありません。
- **視線誘導を計算したインターフェース設計**
  使う人の迷いを無くし、現場の人間が最もスムーズに、無駄なく動けるための地に足のついたUIを構築。
- **安全なRSSフィードを活用した「C案のルート精神」**
  大手サイトの堅牢なセキュリティの壁に対し、強引に突っ込まず（A案・B案を捨て）、公式の安全な窓口である「RSSフィード」を活用。道路という巨大なシステムの中で周囲と調和し、安全に最短で目的地に達するという物流現場のトラック運転で培った最適化ロジックが、この安全ルートを閃かせました。

## 🛠️ システム環境と修得技術

- **言語・フレームワーク:** Python / Flask (マイクロフレームワーク)
- **コアロジック:** RSSフィード解析（WAFやIPブロックを回避する協調型スクレイピング）
- **AI連携:** 100件レベルの最新記事を10秒で処理する高速テキスト要約ロジック
- **フロントエンド:** HTML / Jinja2テンプレート / JavaScript / 視線誘導を計算した実戦的UI設計
- **開発環境:** Ubuntu (Lubuntu) / VS Code / .venv (仮想環境)

## 🔧 『Puoppo』が実現する未来：テレビや新聞を超える情報密度

「知りたい最新情報はたくさんあるけれど、ポチポチ探す時間も、ダラダラとコメンテーターの意見を聴く時間もない。」

Puoppoは、そんな現代人のボトルネックを解消するために生まれました。
あなたが打ち込んだキーワードから、AIがフラットで濃厚な最新情報を一瞬でスマホやPCに叩き込みます。

このシステムを一度体感すれば、もう朝のテレビニュースに戻ることはできなくなるでしょう。
Puoppoによって浮いたその時間で、あなたなら何をしますか？
