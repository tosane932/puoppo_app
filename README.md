## 🛠️ システム環境と修得技術

- **言語・フレームワーク:** Python / Flask (マイクロフレームワーク)
- **コアロジック:** RSSフィード解析（WAFやIPブロックを回避する協調型スクレイピング）
- **AI連携:** 100件レベルの最新記事を10秒で処理する高速テキスト要約ロジック
- **フロントエンド:** HTML / Jinja2テンプレート / JavaScript / 視線誘導を計算した実戦的UI設計
- **開発環境:** Ubuntu (Lubuntu) / VS Code / .venv (仮想環境)

## 🔧 『Puoppo』が実現する未来：テレビや新聞を超える情報密度

「知りたい最新情報はたくさんあるけれど、ポチポチ探す時間も、ダラダラとコメンテーターの意見を聴く時間もない。」

Puoppoは、そんな現代人のボトルネックを解消するために生まれました。
あなたが打ち込んだキーワードから、AIがフラットで濃厚な最新情報を一瞬でスマホやPCに叩き込みます。

このシステムを一度体感すれば、もう朝のテレビニュースに戻ることはできなくなるでしょう。
Puoppoによって浮いたその時間で、あなたなら何をしますか？
